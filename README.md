# gnc-java-final-assessment-smartcampus
Java final assessment
#Student Management System (Java)

##Features

*  Add Students
*  Add Courses
*  Enroll Students
*  View Students
*  View Enrollments
*  Save Data to File
*  Exception Handling
*  Multithreading for Enrollment

##Tech Stack

*  Java (Core)
*  Collections Framework (HashMap, ArrayList)
*  Multithreading
*  File Handling
*  Custom Exceptions

## Project Structure

```bash
StudentManagementSystem.java
│
├── Student
├── Course
├── EnrollmentProcessor (Thread)
└── InvalidDataException
```

### ▶️ Run Locally

```bash
javac StudentManagementSystem.java
java StudentManagementSystem
```
